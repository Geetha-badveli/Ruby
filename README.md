# Ruby
topics of ruby on rails
# Day-1
## 1. Install Ruby(Windows)
* Go to rubyinstaller.org and download the latest Ruby+Devkit version for your system.
* Run downloaded .exe file
* click 1 or 3 in last step or click enter. <br>

  <strong> ---- Ruby will be successfuly installed ----</strong> <br>
<strong> to check ruby is installed properly or not </strong>
* create a file with .rb extention (Ex :- demo.rb)
* write some code by using puts statement(Ex:- puts "hii every one." ).
* Command to run the file(open command prompt(cmd) where file is saved.) <br>
       <strong> ruby file_name.rb </strong>

## 2. Install rails
* Open terminal(cmd) in administrater mode and run command <br>
       <strong> gem install rails </strong>

## 3. Creating Application
* create a folder where you want to create your application
* open cmd in that folder write this command <br>
       <strong> rails new application_name </strong>
* enter into the application folder by using the command. <br>
      <strong> cd application_name </strong>
* run the server by using the command
      <strong> rails s</strong><br>
  If  it opens rails server home page than it is successfuly installed.

  NOTE : download any database for your application. (Postgres sql)
